Lista ordenada

Assim é uma lista ordenada:

  1. O elemento "ol" envolve todos os elementos da lista
  2. Os elementos li continuam representando os itens
  3. Também existem números crescentes representando cada item
  4. Os números aparecem em ordem precisa

Agora criaremos uma lista não ordenada:

  • Adicione um elemento "ul" (este cria uma lista não ordenada)
  • Adicione um item para a lista com o elemento "li"
  • Esta lista não tem nenhuma ordem
