# Classes do BootStrap 4

- resources/clientes/index.blade.php

## Irei coomentar aqui somente as classes do BootStrap usadas nas views principais

Este arquivo começa com:
```php
    container  - container adiciona um padding ao redor de um elemento de forma responsiva
    row  - row cria uma linha para a grid. Sempre antes de adicionar as colunas antes adicionamos uma row

    col-md-9  - col-md-9 aqui uma coluna da grid, com 9 de um total de 12 colunas
    card  - é um box com bordas com algum padding ao redor. Ele inclui opções como headers, footers, content, colors, etc.
    <i class="fa fa-plus" aria-hidden="true"></i>  - ícone de + para o botão que adiciona registro

    accept-charset="UTF-8"  - orienta navegador para aceitar UTF-8

    class="form-inline  - formulário em linha
    my-2  - margem com 2 pixel
    my-lg-0  - margem com telas grandes mas com 0pixel
    float-right  - flutuar à direita
    role="search"  - tem função apenas semântica, para orientar os navegadores

    class="input-group"  - um contêiner para aprimorar uma entrada adicionando um ícone, texto ou um botão na frente ou atrás do campo de entrada como um "texto de ajuda".

    form-control  - classe adicionada em cada input para aplicar o estilo do bootstrap ao mesmo

    input-group-append - para adicionar o texto ao redor do imput

    btn btn-secondary - Aplicar o estilo de botão, no caso verde

    table-responsive  - classe para que tabela fiquem responsivas

    table - classe com o estilo do Bootstrap para tabelas

    btn btn-info btn-sm - botão verde claro e pequeno (sm - small)

    aria-hidden - Indica que o elemento e todos os seus descendentes não são visíveis ou perceptíveis para qualquer usuário, conforme implementado pelo autor.

    pagination-wrapper - classe para os links da paginação


create.blade.php

card-header - card para o header

card-body - para o body

alert alert-danger - escreve o texto de alerta em vermelho

form-horizontal - cria um form na horizontal


form.blade.php

form-group - classe que agrupa um label e um input num form
```
