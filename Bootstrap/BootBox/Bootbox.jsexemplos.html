Examples

Currently viewing examples for version 4.x.

The examples below attempt to demonstrate the myriad ways in which you can use Bootbox.js. Where functionality amongst the dialogs is nearly identical, the example will indicate to which functions the example applies.

Documentation |

Alert

  • Basic usage

    bootbox.alert("This is the default alert!");

  • Basic usage, with callback

    bootbox.alert("This is an alert with a callback!", function(){ console.log('This was logged in the callback!'); });

  • Basic usage, using options object

    1. bootbox.alert({
    2.     message: "This is an alert with a callback!",
    3.     callback: function () {
    4.         console.log('This was logged in the callback!');
    5.     }
    6. })

  • Also applies to: Confirm, Prompt, Custom Small dialog 

    1. bootbox.alert({
    2.     message: "This is the small alert!",
    3.     size: 'small'
    4. });

  • Also applies to: Confirm, Prompt, Custom Large dialog 

    1. bootbox.alert({
    2.     message: "This is the large alert!",
    3.     size: 'large'
    4. });

  • Also applies to: Confirm, Prompt, Custom Custom CSS class 

    1. bootbox.alert({
    2.     message: "This is an alert with an additional class!",
    3.     className: 'bb-alternate-modal'
    4. });

  • Also applies to: Confirm, Prompt, Custom Dismiss with overlay click 

    1. bootbox.alert({
    2.     message: "This alert can be dismissed by clicking on the background!",
    3.     backdrop: true
    4. });

Confirm

  • Basic usage

    bootbox.confirm("This is the default confirm!", function(result){ console.log('This was logged in the callback: ' + result); });

  • Also applies to: Alert, Prompt, Custom With alternate button text and color 

    1. bootbox.confirm({
    2.     message: "This is a confirm with custom button text and color! Do you like it?",
    3.     buttons: {
    4.         confirm: {
    5.             label: 'Yes',
    6.             className: 'btn-success'
    7.         },
    8.         cancel: {
    9.             label: 'No',
    10.             className: 'btn-danger'
    11.         }
    12.     },
    13.     callback: function (result) {
    14.         console.log('This was logged in the callback: ' + result);
    15.     }
    16. });

  • Also applies to: Alert, Prompt, Custom With icon and button text 

    1. bootbox.confirm({
    2.     title: "Destroy planet?",
    3.     message: "Do you want to activate the Deathstar now? This cannot be undone.",
    4.     buttons: {
    5.         cancel: {
    6.             label: '<i class="fa fa-times"></i> Cancel'
    7.         },

    9.             label: '<i class="fa fa-check"></i> Confirm'
    10.         }
    11.     },
    12.     callback: function (result) {
    13.         console.log('This was logged in the callback: ' + result);

    15. });

Custom Dialogs

Please note: Custom dialogs accept only one argument: an options object. The only required property of the options object is message.

  • Custom dialog as "loading" overlay

    1. var dialog = bootbox.dialog({
    2.     message: '<p class="text-center">Please wait while we do something...</p>',
    3.     closeButton: false
    4. });
    5. // do something in the background
    6. dialog.modal('hide');

  • Also applies to: Alert, Confirm, Prompt Custom dialog, using init 

    1. var dialog = bootbox.dialog({
    2.     title: 'A custom dialog with init',
    3.     message: '<p><i class="fa fa-spin fa-spinner"></i> Loading...</p>'
    4. });
    5. dialog.init(function(){
    6.     setTimeout(function(){
    7.         dialog.find('.bootbox-body').html('I was loaded after the dialog was shown!');
    8.     }, 3000);
    9. });

  • Custom dialog, with buttons and button callbacks

    1. var dialog = bootbox.dialog({
    2. title: 'A custom dialog with buttons and callbacks',
    3. message: "<p>This dialog has buttons. Each button has it's own callback function.</p>",
    4. buttons: {
    5.     cancel: {
    6.         label: "I'm a custom cancel button!",
    7.         className: 'btn-danger',
    8.         callback: function(){
    9.             Example.show('Custom cancel clicked');
    10.         }
    11.     },
    12.     noclose: {
    13.         label: "I'm a custom button, but I don't close the modal!",
    14.         className: 'btn-warning',
    15.         callback: function(){
    16.             Example.show('Custom button clicked');
    17.             return false;
    18.         }
    19.     },
    20.     ok: {
    21.         label: "I'm a custom OK button!",
    22.         className: 'btn-info',
    23.         callback: function(){
    24.             Example.show('Custom OK clicked');
    25.         }
    26.     }
    27. }
    28. });
